### JOB COIN:

This is a project using the JobCoin API to fetch, display, and send job coins.

### Install
```
npm install
```

### Dependencies
* React
* Chartjs
* Bootstrap
* Express
* Moment
* jQuery
* Webpack
* Babel

### Start servers
```
npm run server-dev
npm run react-dev
```

### Go to
```
http://localhost:3000
```
